## üìå Course Coverage

The program combines daily theory, challenges, and real apps to provide hands-on experience with:

- **Swift language fundamentals** ‚Äî optionals, protocols, extensions, closures, generics, error handling.  
- **SwiftUI essentials** ‚Äî layouts, state management (`@State`, `@Binding`, `@ObservedObject`), forms, navigation, lists.  
- **Persistence** ‚Äî `UserDefaults`, Codable, and local data storage.  
- **Networking** ‚Äî working with `URLSession`, JSON decoding, and building API-driven forms.  
- **Architecture** ‚Äî MVVM patterns using SwiftUI property wrappers and data flow.  
- **iOS ecosystem** ‚Äî alerts, gestures, animations, haptics, and integration with system features.
